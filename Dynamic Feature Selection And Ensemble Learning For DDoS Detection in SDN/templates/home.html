How to Use the System
Using the Notebook

The Notebook page allows you to input network traffic data for analysis:

  1. Navigate to the Notebook page
  2. Fill in the required network traffic parameters
  3. Click "Predict" to analyze the traffic
  4. View results showing whether the traffic is normal or potentially malicious
Understanding the Parameters
  • Service: Type of network service (e.g., http, ftp)
  • Flag: TCP flag
  • Src-Bytes: Bytes sent from source to destination
  • Dst-Bytes: Bytes sent from destination to source
  • Count: Number of connections to the same destination in the past 2 seconds
  • Other parameters: Various metrics related to connection patterns
