About Our System

Advanced ML Algorithms

Our system uses state-of-the-art machine learning algorithms including MLP, Passive Aggressive Classifiers, Naive Bayes, and Stacking Ensemble methods to detect network intrusions.

Feature Engineering

We leverage advanced feature selection with Mutual Information and dimensionality reduction with PCA to improve detection accuracy.

Real-time Detection

Our system can process network traffic data in real-time to identify potential threats and attacks.

Based on CICDDOS2019 Dataset

Trained on the comprehensive CICDDOS2019 dataset to ensure high accuracy in detecting various types of DDoS attacks.
